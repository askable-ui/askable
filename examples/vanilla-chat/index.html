How this demo works

  1. Every card and row has a data-askable attribute with structured JSON data.
  2. createAskableContext() watches for focus/click events and tracks which element is active.
  3. ctx.toContext() serializes the focused element's metadata into a prompt-ready string.
  4. The mock AI uses that string to answer questions about exactly what you selected.
